
Voting system for movienights

Features to be implemented:
Register
Login (encrypted pwd)
users votes with integer 1 to 5
View previous movie nights with voting statistics and attendees
Users have "star-state" wich weighs their vote more
Users can pull back vote
Admins can create movienights and add movies.
Admins can verify attendees to add to star-state
Admins can cancel movienights
